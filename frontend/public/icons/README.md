# PWA Icons Directory

This directory contains the app icons for the Progressive Web App (PWA).

## Required Icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons:

You can use online tools to generate all required sizes from a single high-resolution image:

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
2. **RealFaviconGenerator**: https://realfavicongenerator.net/
3. **Favicon.io**: https://favicon.io/

## Recommended Source Image:
- Size: At least 512x512px
- Format: PNG with transparent background
- Content: FarmWise logo with the leaf/plant icon

## Quick Setup:
1. Create a 512x512px icon with your logo
2. Use one of the tools above to generate all sizes
3. Place all generated icons in this directory
4. The manifest.json is already configured to use these icons
